# 💜 K-pop Bias Themed To-Do App

An aesthetic and interactive **To-Do List Web App** inspired by K-pop vibes ✨
Choose your favorite group, manage tasks, and enjoy a fun, animated experience!

---

## 🎯 Features

✨ Choose your favorite K-pop group:

* BTS 💜
* Stray Kids 🔥
* Enhypen 💙
* TXT 💛

✨ Dynamic UI:

* Background changes based on selected group
* Custom color themes for each bias

✨ Task Management:

* Add tasks easily
* Press **Enter** or click button
* Tasks saved using **Local Storage**

✨ Fun Completion Effects:

* Random K-pop themed emojis 🎶
* GIF pops up when task is completed 🎬
* Task disappears after a few seconds
